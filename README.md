## Deployment

* Auto deploys are almost setup via codeship and gitlab

## Setup

* `git clone` this repo
* `npm i`
* `npm i -g nodemon` for development

## Running

* run `npm start`
  * open browser to http://localhost:3000

## Development

* run `npm run dev`
  * open browser to http://localhost:3000
    * code changes will auto refresh server
